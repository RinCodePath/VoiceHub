# 🎙️ VoiceHub: Локальный Голосовой Ассистент

![Python](https://img.shields.io/badge/python-3.8+-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Platform](https://img.shields.io/badge/platform-Linux%20%7C%20Windows%20%7C%20macOS-lightgrey.svg)

**VoiceHub** — это быстрый, легковесный и полностью конфиденциальный голосовой ассистент с современным графическим интерфейсом на базе `Tkinter`. Приложение работает **на 100% локально** и автономно (без интернета), используя движок распознавания речи **Vosk**. Ваш голос не отправляется на сторонние сервера или в облака.

Проект разработан с учетом кроссплатформенности и одинаково стабильно работает на **Linux (Arch Linux, Ubuntu, Debian)**, **Windows** и **macOS**.

---

## ✨ Основные возможности

* 🌐 **Кроссплатформенный запуск браузера:** Автоматически подстраивается под системные утилиты (`xdg-open`, `start`, `open`). На Linux имеет встроенный фолбэк на принудительный запуск `firefox`.
* 💻 **Интеграция с PyCharm:** Интеллектуальный поиск путей для запуска IDE PyCharm (Community / Professional) в зависимости от типа установки на Linux/Windows/macOS.
* 🎮 **Запуск Steam / Игр:** Быстрый вызов игрового клиента (`steam`) или запуск исполняемых файлов по кастомным путям.
* 🛡️ **Умная обработка ошибок (Отказоустойчивость):** Никаких безмолвных падений фонового потока. Приключение перехватывает любые системные сбои (неверный путь в конфиге, отсутствие программы в системе) и мгновенно выводит предупреждение красным цветом прямо в основное окно GUI, сохраняя полную работоспособность.
* 🔄 **Умное переключение языков:** Поддерживает работу с русским и английским языками. Встроена жесткая проверка: если вы дадите команду сменить язык, но на диске отсутствует папка `model/en`, ассистент выдаст предупреждение и **не уронит приложение**, продолжив работу на русском.
* 🎨 **Интерфейс в стиле Catppuccin Mocha:** Тёмная, приятная для глаз цветовая палитра интерфейса, разработанная на чистом `Tkinter / ttk` без тяжелых внешних библиотек.

## 🏗️ Архитектура проекта

Структура папок спроектирована по модульному принципу:

```text
VoiceHub/
├── model/                  # Папка для локальных моделей Vosk

│   ├── commands.py         # Словари голосовых фраз и триггеров

Ошибка ALSA lib ... Unknown PCM или треск в консоли на LinuxЭто стандартные информационные сообщения звуковой подсистемы ALSA при работе с PyAudio. Они не влияют на работу программы. Если ассистент реагирует на голос — всё в порядке.2. Ошибка критического запуска Model folder not foundПрограмма не может найти папку model/ru или model/en. Убедитесь, что вы распаковали архив модели Vosk, и структура папок выглядит именно так: model/ru/README, model/ru/am, model/ru/conf и т.д.3. Статус загорается красным: ❌ Не удалось открыть редактор 'pycharm'Это означает, что PyCharm установлен в вашей ОС нестандартным способом (например, через JetBrains Toolbox или кастомный архив без создания симлинков в PATH).Решение: Узнайте, какой командой запускается ваш PyCharm из обычного терминала, и пропишите это точное имя в config.json в поле "editor".📄 ЛицензияПроект распространяется под открытой лицензией MIT.
